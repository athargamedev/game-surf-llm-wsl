# Changelog

All notable changes to the GameSurf NPC Kit will be documented in this file.

## [0.1.0] - 2026-05-02

### Added
- `NpcProfile` ScriptableObject for NPC character configuration
- `NpcDialogueController` MonoBehaviour for dialogue lifecycle management
- `NpcMemoryManager` for Supabase-backed memory persistence
- `ResponseCleaner` utility ported from Python server logic
- `SupabaseConfig` ScriptableObject for connection settings
- Custom inspector for NpcProfile with prompt preview and LoRA status
- Remote server mode (proxy to LLM WSL integrated server)
- Basic NPC Chat sample scene (placeholder)
- Memory Persistence sample scene (placeholder)
